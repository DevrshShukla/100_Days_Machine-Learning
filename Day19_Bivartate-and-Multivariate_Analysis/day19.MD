## 6Ô∏è‚É£ Scatter Plot (Numerical vs Numerical)

### Concept Used

**Bivariate Analysis**

### Purpose

Used to analyze the relationship between **two numerical variables**.

### Example Insight

In Titanic or Tips dataset:

* Total bill vs Tip amount
* Age vs Fare

If points show an upward trend ‚Üí positive correlation.

## 7Ô∏è‚É£ Bar Plot (Categorical vs Numerical)

### Concept Used

**Bivariate Analysis**

### Purpose

* Shows average or total numerical values per category

### Example

* Average fare per passenger class
* Average tip by day

Bar height represents aggregated statistics like mean or sum.

---

## 8Ô∏è‚É£ Box Plot (Categorical vs Numerical)

### Concept Used

**Bivariate Analysis**

### Purpose

* Shows data distribution
* Identifies outliers
* Compares spread across categories

### Example

* Fare distribution by passenger class

Box plot components:

* Median
* Quartiles
* Whiskers
* Outliers

**[ Numerical - Numerical ]**

### Code Cell

```python

```

### Explanation

* **Multivariate Analysis tool**
* Plots every numerical feature against others
* Color (`hue`) separates categories

Used to:

* Identify feature separability
* Detect correlations
* Check class overlap

Very useful before classification models.
